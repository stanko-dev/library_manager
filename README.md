# 📚 Library Manager (Spring Boot + MongoDB)

Цей проєкт реалізує просте CRUD-застосування для керування бібліотекою з можливістю фільтрації даних.

## Стек технологій
* **Backend:** Java 17, Spring Boot 3.x, Spring Web, Spring Data MongoDB.
* **Database:** MongoDB.
* **Build Tool:** Maven.
* **Frontend:** Чистий HTML5 + JavaScript (Fetch API) + CSS.

## Функціонал
* **CRUD:** Додавання, перегляд всіх, видалення книг.
* **Фільтрація:** Пошук за автором (частковий збіг, без урахування регістру) та фільтрація за жанром.

## 🚀 Інструкція із запуску

### 1. Запуск MongoDB
Переконайтеся, що локальний сервер MongoDB запущено на стандартному порту **27017**.
* **Налаштування:** База даних `library_db` буде створена автоматично Spring Boot.

### 2. Запуск Backend
1.  Відкрийте проєкт у IntelliJ IDEA (або іншому IDE).
2.  Переконайтеся, що всі Maven-залежності завантажені (Reload Maven Project).
3.  Запустіть головний клас `LibraryApplication.java`.
4.  Сервер запуститься на **http://localhost:8080**.

### 3. Запуск Frontend
Інтерфейс доступний автоматично за адресою: **http://localhost:8080**

## 🌐 Приклади REST API маршрутів
| Метод | Маршрут | Опис |
| :--- | :--- | :--- |
| **GET** | `/api/books` | Отримати весь список книг |
| **POST** | `/api/books` | Додати нову книгу (JSON в тілі запиту) |
| **DELETE** | `/api/books/{id}` | Видалити книгу за ID |
| **GET** | `/api/books/search?author={name}` | Пошук за автором (частковий збіг) |
| **GET** | `/api/books/filter?genre={genre}` | Фільтрація за жанром |
